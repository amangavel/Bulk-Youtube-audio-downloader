1. Add all the links of youtube videos that you want to download as audio in url_list.txt file.
	>> Note: different video links must be in different line (don't add all links in one line, press enter before adding other link)
2. Double click on "Youtube audio downloader.bat".
3. It will ask for a folder name in cmd window which is the name of the folder where all the audio files will be stored.
	>> eg. you can give folder name as 'Songs'.
4. After completion cmd window will read 'Press any key to continue...'. Press enter and you are done!
5. Enjoy :)
